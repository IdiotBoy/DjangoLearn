## Django

- 安装
	- pip install Django==1.7

- 创建项目
	- django-admin.py startproject mysite

- 运行
	- python manage.py runserver

- 同步数据库与 model
	- python manage.py makemigrations
	- python manage.py syncdb

- 创建应用
	- python manage.py startapp polls
